## 👨‍💼 ADMIN QUICK GUIDE

### ➤ Creating a New Order (3 Steps)
1. **Dashboard** → **Manage Orders** → **Create Order**
2. Fill in:
   - Recipient name & phone
   - Pickup address (where to collect)
   - Delivery address (final destination)
   - Cost (delivery fee in ₦)
   - Any special notes
3. Click **Save** → System generates tracking ID automatically

### ➤ Assigning Orders to Riders
1. Go to **Orders Management**
2. Find the order (status: Pending)
3. Click **"Assign Rider"** button
4. Select rider from dropdown
5. Confirm → Rider gets notified immediately

### ➤ Creating a Rider Account
1. **Dashboard** → **Riders** → **Create Rider**
2. Enter: Name, Email, Password, Phone
3. Save → Rider can now login with provided credentials

### ➤ Marking Orders as Paid
1. Find order in **Orders** list
2. Click **"Mark as Paid"** button
3. Confirm → Payment recorded with timestamp

## 🏍️ RIDER QUICK GUIDE

### ➤ Viewing Your Assigned Orders
1. Login → **Dashboard** shows summary
2. Click **"View All Orders"** or go to **My Deliveries**
3. See all orders assigned to you with status

### ➤ Updating Order Status (Complete Workflow)

**Follow these steps in order:**

#### Step 1: You're Assigned
- Status shows: **"Assigned"**
- Review order details, pickup & delivery addresses

#### Step 2: Collect the Package
1. Go to pickup address
2. Collect package from pharmacy/supplier
3. Open order → Change status to **"Picked Up"**
4. Click **Update Status**

#### Step 3: Deliver the Package
1. Go to delivery address
2. Change status to **"In Transit"** (while en route)
3. Click **Update Status**

#### Step 4: Complete Delivery
1. Hand package to recipient
2. Change status to **"Delivered"**
3. Click **Update Status**
4. If payment needed → Mark as Paid (see below)

### ➤ Collecting & Recording Payment
**When order is Cash on Delivery (COD):**

1. Check delivery cost on screen (₦X.XX)
2. Collect cash from recipient
3. Click **"Mark as Paid"** button
4. Confirm → Payment recorded
5. Remit cash to admin per company policy

### ➤ Using Maps for Navigation
- On order detail page
- Click **"Open in Maps"** next to address
- Opens in Google Maps for easy navigation

## 📊 ORDER STATUS EXPLAINED

| Status | Meaning | Who Updates |
|--------|---------|-------------|
| 🟡 **Pending** | Just created, no rider yet | - |
| 🟠 **Assigned** | Rider assigned, ready for pickup | Admin |
| 🔵 **Picked Up** | Package collected from pickup location | Rider |
| 🟣 **In Transit** | On the way to delivery address | Rider |
| 🟢 **Delivered** | Successfully delivered | Rider |
| 🔴 **Cancelled** | Order cancelled | Admin |

## 🆘 COMMON QUESTIONS

**Q: I can't see my assigned order**  
A: Refresh the page or log out and back in. Check notifications.

**Q: How do I contact the recipient?**  
A: Click the phone number on order detail page to call directly.

**Q: Can I edit an order after creating it?**  
A: Yes (Admin), go to Orders → Click Edit icon (pencil).

**Q: What if delivery address is wrong?**  
A: Contact admin immediately. Admin can edit order details.

**Q: When should I mark as paid?**  
A: Only after physically receiving cash from recipient.

**Q: Can I see my past deliveries?**  
A: Yes, go to My Deliveries → Filter by "Delivered" status.
